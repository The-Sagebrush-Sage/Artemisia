СОГЛАШЕНИЕ

1. Общие положения

1.1. Настоящее Соглашение регулирует отношения между администрацией сайта Artemisia (далее — «Платформа») и пользователем (далее — «Пользователь»).

1.2. Использование Платформы означает полное согласие с условиями Соглашения.

1.3. Платформа вправе изменять условия Соглашения и Правил с публикацией новой версии на сайте.

2. Термины

2.1. Гарант — представитель Платформы, обеспечивающий безопасность сделок между пользователями.

2.2. Арбитраж — процедура разрешения споров Гарантом.

2.3. ТЗ (Техническое Задание) — требования Заказчика к работе, включая сроки, стиль и условия правок.

COMMISSION ART - КММ - КОММИШКА - КОММИШЕН - арт на заказ

YCH - ЮЧ - шаблон с определённой позой персонажа/персонажей, по которому художник по заказу клиента создаёт полноценный арт

ADOPT - АДОПТ - персонаж на продажу

3. Условия использования

3.1. Пользователь обязуется:

  • Не нарушать авторские права
  • Не размещать незаконный контент
  • Не использовать Платформу для мошенничества

3.2. Платформа вправе удалять контент и блокировать аккаунты за нарушения.

4. Гарант сделок

4.1. Услуга Гаранта доступна для:

  • Заказа художественных работ
  • Покупки цифровых товаров

4.2. Гарант:

  • Фиксирует условия сделки (сумму, сроки, ТЗ)
  • Проверяет выполнение обязательств
  • Решает споры через арбитраж

4.3. Комиссия за услугу Гаранта указывается при создании сделки

5. Ответственность

5.1. Платформа не несёт ответственности за:

  • Сделки без участия Гаранта
  • Некорректное ТЗ, которое привело к спору

ПРАВИЛА ИСПОЛЬЗОВАНИЯ

1. Услуги

1.1. Запрещено:

  • Продавать, выкладывать и предлагать услуги не связанные с тематикой Платформы

2. Размещение контента

2.1. Разрешено:

  • Оригинальные работы
  • Контент 18+ с маркировкой для Prem

2.2. Запрещено:

  • Плагиат, реклама, экстремизм
  • Обводка
  • Использование ИИ

3. Гарантированные сделки

3.1. Для Заказчиков:

  • Оплата через Гаранта защищает от мошенничества
  • Можно запросить арбитраж при несоответствии работы ТЗ

3.2. Для Исполнителей:

  • Гарант подтверждает оплату перед началом работы
  • Средства перечисляются после приемки

4. Арбитраж

4.1. Основания для арбитража:

  • Нарушение ТЗ
  • Просрочка
  • Плагиат
  • Обводка
  • Использование ИИ

4.2. Доказательства:

  • Переписка
  • Исходные файлы

4.3. Решение Гаранта обязательно к исполнению.

5. Комиссия

5.1. Взимается за услуги Гаранта (5-10% от суммы сделки)

ПРАВИЛА ГАРАНТИЙНЫХ СДЕЛОК

1. Создание сделки

1.1. Обязательно указание:

  • Точной суммы
  • Детального ТЗ
  • Сроков

2. Процесс выполнения

2.1. Исполнитель показывает работу

2.2. Заказчик проверяет соответствие ТЗ

3. Автоприемка

3.1. Если Заказчик не ответил за 3 дня — работа считается принятой

4. Споры

4.1. При арбитраже Гарант учитывает:

  • Чёткость ТЗ
  • Факт нарушения сроков
  • Доказательства сторон
